1. Grab a still frame from last week's live stream local recording
2. Create a new cover slide using Adobe Photoshop. I combine the frame grab from the step above with a Photoshop Template
3. Use the YouTube studio to schedule an upcoming live stream by reusing the settings from the last live stream. Be sure to increment the show number, change the date in the Title, and add a one-line slug to the description. Upload the new cover slide during the stream creation process and be sure to set the start time to the right date and time (considering timezones)
4. Copy the resulting live stream URL and paste it into the "FJHH show notes" google doc.
5. That document contains a template for sending email to the everyone@universitylutheran.church mailing list. Create and schedule that gmail to be sent wednesday morning.
6. Log into mailchimp and create a new campaign by duplicating the most recent one. Update the Date and Show Number in several places. Paste in the one-line slug from step three. Schedule the mailchimp email to be sent wednesday morning.
7. Log into Facebook and create a new Facebook event using the show number, date, time, youtube URL, cover slide, and one-line slug. Facebook suggests some people to invite, so invite them.
8. Log into Bluesky and schedule a post to go out Friday noon containing the youtube URL and the one-line slug.
